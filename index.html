WRCC wea server by gkwrcc

A project to serve data from wea files as JSON over http.

The WRCC stores many weather station data in a custom binary format, internally known as .wea. This is a project to make those data available in a JSON format.

Dependencies

Python, numpy

Install

This probably isn't useful outside of WRCC.
